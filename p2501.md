### 1. 문제분석 & 이유
---
N의 약수들 중 k번째를 찾을것, 없다면 0을 출력


### 2. Pseudo Code

---

```
n과 k를 입력받기

갯수를 담을 count 변수

1부터 n까지 :
나누었을때 0이되면 count+=1 
그러다가 원하는 k번째가 되면 출력하고 멈추기

만약 k보다 count가 작으면 0을 출력
```


### 3. 문제 해결 및 배운 점
---
한번에 성공하였다.

### 4. 최종 코드
---

```
N, K = map(int, input().split())

# N의 약수들 중 k번째의 수 (오름차순) , 만약 null이면 0을 출력
count = 0

for i in range(1,N+1):
    if N % i == 0 :
        count += 1
    if count == K :
        print(i)
        break


if count < K :
    print(0)

```
